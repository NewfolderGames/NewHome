검색문
검색문은 한글로도 작성 가능합니다.
이름과 태그는 한번만 작성 가능합니다.
'<'이나 '>='같은 연산자를 쓰지 않으면 자동으로 '=='으로 바뀝니다.
이름(name)
이름으로 제품을 검색할 때 사용 됩니다.
검색문을 쓰지 않아도 사용 가능합니다.
name:[이름] / name:=[이름] / name:==[이름]
태그(tag)
태그가 붙은 제품을 검색할 때 사용 됩니다.
tag:[태그] / tag:=[태그] / tag:==[태그]]
가격(price)
가격 기준으로 제품을 점색할 때 사용 됩니다.
여러개를 사용할 수 있습니다.
price:<=[가격] / price:[가격] / price:>=[가격]
평가(rating)
평가 기준으로 제품을 점색할 때 사용 됩니다.
여러개를 사용할 수 있습니다.
rating:<=[평가] / rating:[평가] / rating:>=[평가]
